# Planning

Use this guide when a task spans more than two files or has meaningful risk.

- Confirm the requested target with a search or direct read.
- State the goal, exact files, constraints, and one observable completion check.
- Run the narrowest available baseline check before editing.
- Split work when steps can be verified independently.
- Ask only when two reasonable interpretations create materially different outcomes.
- Revise the plan when evidence disproves a premise or scope expands substantially.
