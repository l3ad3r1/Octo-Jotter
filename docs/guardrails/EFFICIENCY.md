# Efficiency

- Locate targets with search, then read only the relevant scope.
- Do not load generated, vendored, lock, or large data files in full.
- Batch independent reads and checks.
- Cap or summarize output expected to exceed 200 lines.
- Reuse fresh evidence instead of repeating searches and reads.
- Delegate only clearly independent, bounded work when delegation is available.
- Keep progress updates brief and decision-focused.
